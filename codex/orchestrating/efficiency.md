# Efficient dispatch contract

Start Caveman ultra and Ponytail ultra for lite, medium, and orchestrator work. Preserve structured internal notes and the two concise lines below. Honor explicit override; use clarity/correctness escalation when compression or minimalism risks safety, requirements, or verification.

Every subagent prompt should carry only:

- objective and owned files;
- relevant path/symbol references;
- exclusions and shared-workspace warning;
- runnable verification;
- return format of at most two concise lines.

Normal live work uses Terra with medium reasoning. Sol is reserved for benchmark sentinels or evidence-backed escalation. Prefer one well-bounded agent over several overlapping agents. Keep raw logs on disk and return conclusions plus evidence.

Detect profile selection from the active tool schema. When no agent-type/profile field exists, include this role contract directly in the dispatch: own only named files; verify named upstream symbols; run the named check; write the named done note; return shipped scope plus verification in two concise lines. Never claim a custom profile was applied when the surface could not select one.
