# RTK - Rust Token Killer

Token-optimized CLI proxy. The Bash hook rewrites shell commands automatically (`git status` → `rtk git status`) — never hand-wrap.

- Meta (use directly): `rtk gain` savings · `rtk discover` missed opportunities · `rtk proxy <cmd>` raw passthrough
- `rtk gain` errors → name collision (Rust Type Kit) or missing binary: `which rtk`
