# INDEX

Router for `claude/design-guidelines/`. Exempt from per-doc validation (see `CONVENTIONS.md`); the doc table below is the single source cross-refs resolve against.

## Load order (JIT)

1. `INDEX.md` (this file) — always first, never skipped.
2. Only the docs the task table routes to, in listed order — never bulk-load the directory.
3. `CONVENTIONS.md` — only when authoring or editing guideline docs themselves.
