# lite.md — post-cycle reset

For right after an orchestrating cycle lands, or any repo with only obvious junk. **One pass, one manifest, <~15 paths.** Routed here from `README.md`.

## Activate

`/caveman:caveman ultra` + `/ponytail:ponytail ultra` + RTK + Serena. Output styles off. Obey `README.md` §Invariants.

## Flow

1. **Preflight.** `rtk git status` — uncommitted real work → commit or stash it first (cleaning commits stay pure). Post-cycle or whenever `.orchestrator/` exists: **read `.orchestrator/state.md` first** (first line = lifecycle marker) — `landed`|`abandoned` → decided: harvest what exists + reset, don't re-ask the gate (`landed`: `git log origin/main..main` nonempty first → the land never pushed, D died between merge and push → push it / flag to user BEFORE deleting artifacts, else main sits silently unshipped); `parked` → offer **D-RESUME** (fresh chat: read `~/.claude/orchestrating/session-d.md` + `.orchestrator/handoff.md` + `progress.md`, re-run the land gate from the recorded objection), keep `.orchestrator/` + unit branches, clean the rest; `running` → **hard git gate FIRST**: `git merge-base --is-ancestor integration main` (true for ff + merge-commit lands both) + no unmerged `<prefix>/*` unit branches → cycle actually LANDED (D crashed before the marker) → treat as the `landed` path (harvest + reset), skip the RESUME/ABANDON/STOP question; only a genuinely-not-landed `running` (run may be live in another window) → AskUserQuestion: **RESUME** (print the orchestrator C resume kickoff — copy it from `~/.claude/orchestrating/session-c.md` §Resume kickoff — exclude `.orchestrator/` + unit branches from this clean, continue with the rest) vs **ABANDON** (harvest whatever `.done.md`s exist, write `state.md` = `state: abandoned — <reason>`, list unmerged branches report-only — user deletes those, never this run) vs **STOP**. **state.md missing** (legacy dir, pre-marker) → **hard git gate**: the cycle actually landed (`git merge-base --is-ancestor integration main` — true for ff + merge-commit lands both; no unmerged `<prefix>/*` unit branches) before touching `.orchestrator/`; **not landed** → the same RESUME/ABANDON/STOP question above.
2. **Gitignore.** Patch `.gitignore` from `structure.md` §gitignore baseline; untrack newly-ignored paths: `git rm -r --cached -q <path>`.
3. **Harvest, then manifest.** Before deleting `.orchestrator/`, harvest durable knowledge (one cheap pass — files are ≤15 lines each), per `structure.md` §harvest table: blocked/ship-without epics → `docs/03-roadmap.md`; `.done.md` `decided:` lines → `docs/decisions/YYYY-MM-DD-<cycle>.md` (ONE file per cycle); `gotchas:` → `docs/02-architecture.md` §Gotchas; `depmap.md` files-owned seams → `docs/02-architecture.md` §Ownership zones (copy only when the globs differ from current zone lines); `cost:`/`process:` lines from `progress.md` → 2-line footer of the cycle's decisions file; `blockers.md` env/infra facts → `docs/04-operations.md`. **Read `structure.md` §Harvest table + §junk table + §gitignore baseline ONLY** (grep headings / offset-limit — never the whole file, ~11KB; lite needs ~4.5KB). **Idempotence guard:** grep each destination for this row's CONTENT (unit/path + trap text — NOT the bare date; two same-day cycles share a date) first — already present (crashed cleaning re-run, or D partially harvested) → skip those rows, don't append duplicates. Then **manifest → approve → execute.** Spent state only, per `structure.md` §junk table: `.orchestrator/` (landed cycle), scratch/tmp/debug files, `.DS_Store`, committed build artifacts, dead handoff/plan files (content shipped — verify via `git log --oneline -5 -- <path>`). Table `action | path | reason` → ONE AskUserQuestion → execute.
4. **Branch prune.** `git branch --merged main | grep -v ' main'` → `git branch -d` each (safe-delete only, never `-D`). Stale *unmerged* branches → list in summary, don't touch. **Orphan worktrees:** `git worktree prune` (dead dispatched agents leave worktrees; nothing else in the lifecycle removes them) — any worktree still listed after prune → list in summary, don't remove.
5. **Land.** `git commit -q` (`chore: post-cycle cleanup`) + caveman summary: deleted / untracked / pruned counts.

**Escalate:** doc sprawl, renames, or a docs refresh needed → `medium.md`. Save the manifest state first.
